---

## Purpose

Scoreo is built for **honest evaluation**.

It eliminates:
- Random practice
- Repeated question banks
- Answer-peeking
- Noise and distraction

Each test is:
- Generated on demand
- Strictly limited to the requested topic
- Difficulty-aware
- Answer-hidden until completion
